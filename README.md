# GuardNet

React + Vite + TypeScript app with Tailwind CSS, Firebase SDK v10, React Leaflet, and PWA support.

## Setup

```bash
npm install
npm run dev
```

## Environment Variables

Create a .env file based on .env.example and provide the following values:

- VITE_OPENWEATHER_KEY
- VITE_MAPBOX_TOKEN
- VITE_FIREBASE_API_KEY
- VITE_FIREBASE_PROJECT_ID
- VITE_ANTHROPIC_KEY
- VITE_ORS_KEY

## Build

```bash
npm run build
```
