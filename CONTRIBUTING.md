# Contributing to GuardNet

Thanks for your interest in contributing!

## Getting Started

1. Fork the repo
2. Create a branch: `git checkout -b feature/your-idea`
3. Install deps: `npm install`
4. Copy `.env.example` to `.env` and fill in keys (optional for demo mode)
5. Run dev: `npm run dev`

## Guidelines

- Keep PRs focused on a single change
- Run `npm run build` before submitting
- Match the existing code style
- Test in both online and offline modes if relevant

## Project Structure

- `src/components/` — React components
- `src/services/` — API and business logic
- `src/hooks/` — Custom React hooks
- `src/utils/` — Utilities (demo data, offline queue)
- `src/types/` — TypeScript type definitions

## Questions?

Open an issue or start a discussion. We're friendly!
